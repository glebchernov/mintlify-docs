# API Salsa as a Service v2.0

L'API v2.0 ajoute le support GraphQL, webhooks améliorés et nouveaux endpoints de prévision IA.

## URL de base

```
https://api.salsaasaservice.com/v2
```

<Info>
**Nouveau dans v2.0** : Endpoint GraphQL disponible sur `https://api.salsaasaservice.com/graphql`
</Info>

## Authentification

Toutes les requêtes API nécessitent une authentification avec votre clé API.

### En-tête d'authentification

<CodeGroup>
```bash cURL
curl -X GET https://api.salsaasaservice.com/v2/ingredients \
  -H "Authorization: Bearer VOTRE_CLE_API"
```

```python Python
import requests

headers = {"Authorization": "Bearer VOTRE_CLE_API"}
response = requests.get(
    "https://api.salsaasaservice.com/v2/ingredients",
    headers=headers
)
```

```javascript JavaScript
const response = await fetch(
    "https://api.salsaasaservice.com/v2/ingredients",
    { headers: { "Authorization": "Bearer VOTRE_CLE_API" } }
);
```
</CodeGroup>

<Warning>
**Changement v2.0** : Les clés API v1.0 ne fonctionnent pas avec v2.0. Générez une nouvelle clé depuis **Paramètres > Développeur > API v2.0**.
</Warning>

## Limites de taux

- **Plan Starter** : 10 000 requêtes par heure (augmenté depuis v1.0)
- **Plan Professionnel** : 100 000 requêtes par heure
- **Plan Entreprise** : Illimité


## Prévisions IA (Nouveau dans v2.0)

### Obtenir les prévisions de demande

<CodeGroup>
```bash cURL
curl -X GET "https://api.salsaasaservice.com/v2/ai/forecast?location=loc_123&days=7" \
  -H "Authorization: Bearer VOTRE_CLE_API"
```

```python Python
response = requests.get(
    "https://api.salsaasaservice.com/v2/ai/forecast",
    headers={"Authorization": "Bearer VOTRE_CLE_API"},
    params={"location": "loc_123", "days": 7}
)
```

```javascript JavaScript
const response = await fetch(
    "https://api.salsaasaservice.com/v2/ai/forecast?location=loc_123&days=7",
    { headers: { "Authorization": "Bearer VOTRE_CLE_API" } }
```

## Webhooks v2 (Amélioré)

<Note>
**Nouveau dans v2.0** : Système de webhooks amélioré avec livraison garantie, filtrage d'événements, vérification de signature et suivi de statut.
</Note>

### Événements disponibles

- `forecast.updated` — Prévisions IA mises à jour
- `sensor.reading` — Nouvelle lecture de capteur
- `sensor.alert` — Alerte de capteur déclenchée
- `production.batch_completed` — Lot de production terminé
- `inventory.low_stock` — Stock faible détecté
- `quality.alert` — Problème de qualité détecté

### Configurer un webhook

```json
{
  "url": "https://votre-app.com/webhooks/salsa",
  "events": [
    "forecast.updated",
    "sensor.alert",
    "production.batch_completed"
  ],
  "secret": "votre_secret_webhook",
  "active": true
}
```
