# Démarrage avec Salsa as a Service v2.0

Bienvenue sur v2.0 ! Ce guide vous aidera à configurer votre compte et à exploiter les nouvelles fonctionnalités alimentées par IA, capteurs IoT et intégrations avancées.

## Prérequis

Avant de commencer, assurez-vous d'avoir :

- Une adresse e-mail professionnelle valide
- Accès aux systèmes de votre restaurant (POS, inventaire, etc.)
- Connaissance de base de vos opérations actuelles de salsa
- Accès administrateur pour configurer les intégrations
- **Nouveau dans v2.0** : Matériel de capteurs IoT optionnel pour surveillance qualité

## Assistant de configuration rapide

<Note>
**Configuration intelligente** : v2.0 inclut un assistant de configuration alimenté par IA qui analyse votre profil de restaurant et recommande une configuration optimale.
</Note>

<Steps>
  <Step title="Créer votre compte">
    Inscrivez-vous avec votre e-mail professionnel. v2.0 offre l'intégration SSO avec Google, Microsoft et Okta pour les clients entreprise.

    <Tip>Les clients entreprise peuvent activer le Single Sign-On lors de l'inscription pour un accès fluide de l'équipe.</Tip>
  </Step>
  <Step title="Configuration de profil intelligent">
    Notre assistant IA analyse votre type de restaurant et recommande les paramètres optimaux.
  </Step>
  <Step title="Connecter vos systèmes">
    v2.0 détecte les systèmes compatibles sur votre réseau et offre une configuration d'intégration en un clic.
  </Step>
  <Step title="Importer les données historiques">
    Importez optionnellement les données historiques de ventes et production pour entraîner le moteur de prévision IA.
  </Step>
  <Step title="Configurer les fonctionnalités IA">
    Activez la prévision de demande, prédiction de qualité et réapprovisionnement automatique.
  </Step>
</Steps>

## Configuration du compte

### Étape 1 : Création de compte avancée

1. Visitez notre [page d'inscription](https://app.salsaasaservice.com/signup)
2. Choisissez votre méthode d'authentification :
   - E-mail/mot de passe avec 2FA
   - SSO (Google, Microsoft, Okta)
   - SAML 2.0 pour entreprise
3. Vérifiez votre e-mail ou fournisseur SSO
4. Complétez la vérification d'entreprise améliorée

<Note>
**Sécurité renforcée** : v2.0 supporte l'authentification biométrique, clés de sécurité matérielles et options de connexion sans mot de passe.
</Note>

## Configuration des fonctionnalités IA

### Configuration de prévision de demande

Activez la prévision de demande alimentée par IA pour prédire les besoins en salsa 7-30 jours à l'avance :

<Steps>
  <Step title="Activer les prévisions">
    Allez dans **Fonctionnalités IA > Prévision de demande** et activez la fonctionnalité.
  </Step>
  <Step title="Connecter les sources de données">
    Liez votre système POS, système de réservation et toutes sources de données externes.
  </Step>
  <Step title="Import de données historiques">
    Importez au moins 90 jours de données de ventes historiques pour une précision optimale.
  </Step>
  <Step title="Définir les paramètres">
    Configurez l'horizon de prévision (7, 14 ou 30 jours) et les seuils d'alerte.
  </Step>
</Steps>

<Note>
**Précision des prévisions** : L'IA atteint une précision de 70-80% avec 90 jours de données et s'améliore à 95%+ après 6 mois.
</Note>

### Configuration des capteurs IoT

<Steps>
  <Step title="Sélection du matériel">
    Choisissez des capteurs compatibles de notre catalogue certifié.
  </Step>
  <Step title="Installation physique">
    Suivez le guide d'installation. Notre application mobile inclut la visualisation AR pour le placement.
  </Step>
  <Step title="Configuration réseau">
    Connectez les capteurs via WiFi, Bluetooth ou notre appareil passerelle IoT.
  </Step>
  <Step title="Enregistrement des capteurs">
    Utilisez l'application mobile pour scanner les codes QR et enregistrer automatiquement.
  </Step>
</Steps>

<Warning>
**Exigences matérielles** : Les capteurs IoT nécessitent une connexion réseau stable. Nous recommandons un réseau IoT dédié pour la sécurité et la fiabilité.
</Warning>
